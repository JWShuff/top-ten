The a element

So, the a element serves to format links and act as a clickable element on a page. It can point to links, emails, even locations in this document!

It also has all sorts of CSS styling options available that alter behavior from hovered, to clicked, and so on.

 < a > is the opening tag and < / a > is the closing tag. < a href= >normally begins the link, and you can use mailto:, #(insertIDInPageHere), or a web link.
